# Warung Tenongan Bu Tutik

A modern landing page created for **Warung Tenongan Bu Tutik**, a local breakfast UMKM located in front of PMI Surakarta.

## Features

- Responsive Design
- Modern UI/UX
- WhatsApp Integration
- Google Maps Integration
- Menu Showcase
- Customer Testimonials
- Business Information
- SEO Friendly
- Meta Pixel Ready

## Technology

- React
- TypeScript
- Tailwind CSS
- Vite
- Framer Motion

## Purpose

This project was developed as part of a Digital Business community service and Meta Ads campaign to help UMKM improve their online presence.
